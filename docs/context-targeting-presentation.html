Quick Wins — Low Complexity
1

Scope-pinned prompts for fan-out agents

Each fan-out child loads only its own step plan, not the full 10-doc set. Router strips unneeded docs before agent invocation. No new infrastructure required.

−62%
2

Doc compression pipeline

One-time pre-run pass: collapse traceability annotations, remove verbatim architecture quotes, deduplicate boilerplate verification sections. One-time effort per run.

−45%
Medium Investment
3

Progressive context assembly

Agents start with compressed ToC (~10K tokens). They fetch specific sections via tool calls only when needed. Adds latency but delivers the biggest overall reduction.

−71%
4

Cross-task context cache

Parent fan-out task builds a structured 5–10K context object. All children inherit shared decisions as structured data, not prose. Scales with parallelism.

−58%
