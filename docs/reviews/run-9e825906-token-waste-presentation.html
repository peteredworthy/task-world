Root Causes

Prompt and routine shape

The parent routine asks for durable Markdown plus machine oversight state, then uses fresh-context builder and verifier phases. That is rigorous, but it is expensive when the phase is mostly bookkeeping.

The task text also told the parent to validate a full orchestration loop and implement phases 5, 6, and gap C. With the named document missing, the agent treated planning and scope recovery as part of the job.

Execution and control flow

The first child was created as user_managed, then paused with no_executor_running. The parent did not short-circuit immediately after that infrastructure state appeared.

Because the parent is required to surface defects and request clarification, it converted the missing executor into another LLM task instead of a low-cost deterministic pending action.

Fixes To Reduce Future Waste

Immediate guardrails

Fail fast when a child starts with user_managed and no executor is attached. This should be a deterministic parent pending action, not a full evidence-evaluation phase.

Require source artifact existence during intake. If a named source document is missing, pause before inventory expansion unless the user explicitly approves a substitute.

Model and phase policy

Reserve Opus for true architecture or ambiguity resolution. Use Haiku or deterministic checks for inventory schema validation, slice template validation, checklist updates, and grade recording.

Collapse verifier passes for parent bookkeeping into mechanical gates where possible: file exists, oversight state updated, child run linked, child status terminal or pending.

Context compression

Pass compact state snapshots into parent phases instead of rehydrating large routine text, repo guidance, and generated Markdown. The ledger already exists; make agents use it as the default control-plane input.

Call batching

Batch checklist updates and grade writes. Avoid repeated get_requirements calls after the task prompt already contains requirements. Prefer one structured submit payload per phase.
