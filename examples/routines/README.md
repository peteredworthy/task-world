# Example Routines

This directory contains example routine templates that demonstrate advanced workflow features.

## Available Examples

- `comprehensive-mcp-tools-example.yaml` — Demonstrates MCP server integration and tool configuration
- `minimal-backward-compat.yaml` — Minimal routine showing backward compatibility
- `step-tools-example.yaml` — Step-level tool configuration

## Production Routines

The production idea-to-plan routine lives at `routines/idea-to-plan/routine.yaml`. See `routines/idea-to-plan/README.md` for usage instructions.
