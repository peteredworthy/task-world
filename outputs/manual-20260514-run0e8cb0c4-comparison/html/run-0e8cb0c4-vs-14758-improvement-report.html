What improved by stage

SP-01 intake became much cheaper. The child evidence/recovery stage became dramatically cheaper because later oversight work used lower-cost profiles and the parent made a direct reject decision for the broken child.

What did not improve by stage

SP-02 slice selection and replacement-child launch were still expensive. The replacement launch consumed 3.22M parent tokens before the run paused on child evidence that still required human review.

Bottom line: there is measurable stage-level improvement in intake and evidence recovery. There is not yet a full-loop efficiency win because replacement-child launch and unresolved child lifecycle state consumed the savings.
